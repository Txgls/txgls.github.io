üìã Commands

Base Commands

  • /classes- opens up the class choosing GUI
  • /pointsshop- opens up the points shop GUI
  • /levels- opens up the levels GUI

Admin Commands

  • /classremove (player)- removes the players class
  • /class-set (player) (class)- sets a player to the specific class
  • /classreload- reloads the editable yaml files for the plugin
  • /class (player)- allows you to see what class a player is in
  • /points add/remove (player) (class) (amount)- allows you to add or remove points to/from a player
  • /pointsview (player) (class)- allows you to view how many points a player has in that class
  • /pointsmultiplier add (player) (class) (amount) (duration)- adds a multiplier to a player (1-5x, duration in seconds)
  • /pointsmultiplier view (player) (class)- views current player multipliers
  • /pointsmultiplier remove (player) (class)- removes multipliers from players
  • /party chatadmin- enables the reviewal of party chats

Party Commands

  • /party create (name)- creates a party
  • /party leave- leaves a party
  • /party view (player)- views a players party
  • /party members (name)- view the members of a party
  • /party kick (player)- can only be used by the leader. Kicks a player
  • /party invite/accept/decline- Invites, accepts or declines invitations
  • /party chat (message)- sends a message in party chat
  • /party warp- teleports to a party warp
  • /party setwarp- creates party warps
  • /party delwarp- deletes party warps
  • /party home- teleports to the party home
  • /party sethome- creates the party home
  • /party delhome- deletes the party home
  • /party list- lists all active parties on the server
  • /party leader- appoints a new leader for the party

Blood Class Commands

  • /bloodlift- enables/disables the blood lift ability
  • /bloodshield- activates the blood shield ability
  • /bloodfrenzy- enables the blood frenzy ability

Water Class Commands

  • /watershield- activates the water shield ability
  • /hydrodash- enables/disables the hydro dash ability
  • /aquasurge- enables the aqua surge master ability

Fire Class Commands

  • /fireshield- activates the fire shield ability
  • /scorchedearth- enables/disables the scorched earth ability
  • /firerage- enables the fire rage master ability

Soul Class Commands

  • /soulshield- activates the soul shield ability
  • /soulsteal- enables/disables the soul steal ability
  • /soulcrush- activates the soul crush master ability

Snow Class Commands

  • /snowshield- activates the snow shield ability
  • /proximityfreeze- enables/disables the proximity freeze ability
  • /blizzard- activates the blizzard ability

Lightning Class Commands

  • /electricstorm- activates the electric storm ability
  • /electricshield- activates the electric shield
  • /lightningdash- activates the lightning dash ability
