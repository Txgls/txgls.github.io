MythicLifesteal

Heart-Based Survival System

Very configurable lifesteal plugin where players lose hearts on death and gain them by killing others. Work smarter, not harder - take a break, you deserve it. Fully commented configuration files make setup a breeze.

How It Works

1

Start with Hearts

Players begin with configurable heart count (default 10). You can be cruel and set it to 1 if you want!

2

Death Loses Hearts

When a player dies, lol loser, they lose a heart. Only way to get hearts back is by killing others or trading.

3

PvP Steals Hearts

Kill another player to steal their heart! The victim loses a heart while the killer gains one.

4

Trade Heart Items

Withdraw hearts as items to trade with friends. Beg them for hearts or be generous and share!

📋 Commands

Player Commands

  • /mythiclifesteal hearts [player]- Views your or another player's hearts
  • /mythiclifesteal withdraw <amount>- Withdraws hearts as physical items
  • /mythiclifesteal help- Shows the help menu with all commands

Admin Commands

  • /mythiclifesteal sethearts <player> <amount>- Sets a player's heart count
  • /mythiclifesteal addhearts <player> <amount>- Adds hearts to a player
  • /mythiclifesteal removehearts <player> <amount>- Removes hearts from a player
  • /mythiclifesteal reload- Reloads the plugin configuration

💡 Command Aliases

All commands support short aliases: mls, ls, and mythic work the same as /mythiclifesteal!

🔐 Permissions

  • lifesteal.use- Allows basic lifesteal commands (default: true)
  • lifesteal.hearts- Allows checking own hearts (default: true)
  • lifesteal.hearts.others- Allows checking other players' hearts (default: op)
  • lifesteal.withdraw- Allows withdrawing hearts as items (default: true)
  • lifesteal.admin- Allows all administrative commands (default: op)
  • lifesteal.ban.(number)- Replace (number) with a value between 1-24. Will ban the user for that amount of time if bans are enabled.

🛡️ Permission Hierarchy

Most permissions default to true for players, while admin commands require operator status. Perfect for controlled heart economies!
